# Analizator Emocji AI 🧠
Mój pierwszy projekt wykorzystujący modele typu Transformer do analizy sentymentu tekstu.

## Jak to działa? 🔍
Program prosi użytkownika o wpisanie tekstu, a następnie używa modelu od Hugging Face, aby ocenić, czy wypowiedź jest pozytywna czy negatywna.

## Jak uruchomić? 🚀
1. Zainstaluj biblioteki: `pip install transformers torch`
2. Uruchom skrypt: `python My_1st_AI_Project.py`
