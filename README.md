# AI emotion analyzer 🧠

My first project using Transformer type moedls to analyze emotions of given text.

## How it works? 🔍
Program asks user to type text and then uses model from Huggin Face to judge is the text positive or negative.

## How to run it? 🚀
1. Install libraries: `pip install transformers torch`
2. Run the script: `python My_1st_AI_Project.py`

Project has been made in educational purpouses with AI support (Gemini) in order to learn basics of Python and 
Machine Learning libraries.
